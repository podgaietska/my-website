LOTION

Lotion is a web application which resembles the Notes App. Lotion was developed with the use of React (front-end), and Terraform, which was used to build AWS infrustructures, including Lambda functions and DynamoDB table (back-end).

MINI PROJECT

LIBCODE

A library of data structures (developed with Python), designed to be compiled and imported into future projects to make use of the data structure implementations. Included data structures: BST, AVL, Stack, Queue, and a variety of Linked Lists.
BACKEND PROJECT

SCHEDULE BUILDER

An application developed in Java, which provides an automated way to create a schedule for a Wildlife Rescue Center. Is able to analyzes information provided in an SQL file, such as the types of animals present, prescribed treatments, feeding times, etc. and assign tasks hour by hour in an efficient manner.
BACKEND PROJECT

MUSEUM DATABASE

A project which involved designing a relational database with the use of MySQL, to store informations about paintings, artists and hosted exhibitions.
